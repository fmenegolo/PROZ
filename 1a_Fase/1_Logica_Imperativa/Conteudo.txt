

Lógica Imperativa
O uso do raciocínio lógico em situações-problemas

- Lógica
A lógica permite que o ser humano consiga fazer uso do raciocínio lógico para realizar atividades,
sejam elas complexas ou não. Dessa forma, várias ações que realizamos no dia a dia estão relacionadas
à lógica.




- Lógica e diálogo

A comunicação é algo natural para os seres humanos, e, para que ela aconteça, é preciso que haja uma troca de mensagens entre um emissor e um receptor.

Para garantir que essas mensagens sejam coerentes, elas devem seguir uma estrutura lógica que ajude na ordenação dos pensamentos e na melhor compreensão das informações presentes no diálogo.

Sem a presença de algum elemento lógico, é provável que essa troca não faça sentido e acabe comprometendo a comunicação.

"O diálogo é uma troca lógica para que a mensagem seja compreensível."

- Raciocínio lógico

Nós usamos o raciocínio lógico para resolver vários problemas. Por exemplo, ao trocar uma lâmpada queimada, precisamos seguir uma sequência de instruções lógicas, como:

    1º passo — pegar uma lâmpada nova;

    2º passo— pegar a escada;

    3º passo— posicionar a escada embaixo da lâmpada queimada;

    4º passo — subir na escada com a lâmpada nova;

    5º passo — retirar a lâmpada queimada;

    6º passo— colocar a lâmpada nova;

    7º passo — descer da escada;

    8º passo — guardar a escada;

    9º passo — descartar a lâmpada queimada.

Tudo que fazemos no dia a dia está ligado à lógica, pois é preciso utilizar o raciocínio lógico para resolver problemas, mesmo que sejam simples, por exemplo:

    ao entrar em casa durante a noite, o ambiente estará escuro. Logo, raciocinamos que é preciso acender a luz para que o ambiente fique claro;

    ao sair de casa em um dia chuvoso, raciocinamos que é necessário levar um guarda-chuva.
   
    "As atividades diárias precisam ter lógica para serem executadas de forma correta."

- Lógica de programação
    

    Na programação, não é diferente, tudo deve estar baseado em um raciocínio lógico.

    Assim, o principal objetivo da lógica de programação é desenvolver o raciocínio lógico para pensar em como solucionar diversos problemas. 

    Quando fazemos isso, o nosso raciocínio processa a informação em linguagem natural, ou seja, na linguagem que falamos. Com essas informações, podemos criar os algoritmos, que são as instruções necessárias para resolver um problema.

- Raciocínio lógico na programação
    

    No exemplo visto anteriormente, as instruções apresentadas para a troca de uma lâmpada são um algoritmo. Ou seja, existem instruções que formam o caminho a ser seguido para chegar à resolução do problema.

 

    Assim, quando temos um algoritmo construído para resolver um problema, podemos utilizar qualquer linguagem de programação para desenvolver a solução.

- Lógica na programação



    O primeiro passo para programar é aprender a pensar com lógica. Pois, assim, podemos entender o que precisa ser feito e quais são as necessidades para resolver um problema. Dessa forma, é possível compreender como as ações precisam ser realizadas.

 

    Depois de entender a lógica, basta aprender como escrever a solução do problema com uma linguagem de programação adequada.

- Raciocínio lógico
    
    Analisando os exemplos apresentados, não seria possível compreendê-los sem o pensamento lógico, pois ele nos ajuda a entender as problemáticas e a encontrar possíveis soluções a serem testadas.

- Conhecendo a lógica computacional


    Toda tecnologia é desenvolvida para satisfazer as necessidades do ser humano e contribuir para a eficiência das informações.

 

    O computador é uma das tecnologias mais usadas atualmente e a lógica computacional, também conhecida como lógica de programação, foi primordial para o seu avanço.
    
    "Com o passar dos anos, os computadores se tornaram uma ferramenta essencial no dia a dia."


- Lógica computacional


    Com a lógica computacional, é possível implementar comandos para o desenvolvimento de algoritmos e softwares.

 

    Para isso, é necessário que esse raciocínio seja organizado, coerente e relacionado com o conceito do pensamento computacional.
    
    "A lógica é a base para o desenvolvimento de algoritmos."

- Pensamento computacional 

    Blikstein (2008) afirma que o pensamento computacional é manusear o computador de forma criativa e eficaz para solucionar problemas.
   
    Então, essa forma de pensar é uma competência que está baseada na resolução de problemas de forma criativa e está fundamentada em quatro pilares:

        decomposição;
            
            A decomposição é a divisão do problema em porções menores para facilitar a sua resolução.
            
        reconhecimento de padrões;
            
            Já o reconhecimento de padrões é o conjunto de fatos que apontam um problema.

            "Os padrões permitem identificar um problema rapidamente."

        abstração;
            
            A abstração consiste em pensar no problema de maneira genérica, ignorando detalhes que não são importantes para a sua resolução.

        pensamento algorítmico ou simplesmente algoritmo.

            Já o pensamento algorítmico está na sequência de passos para resolver o problema.

    

    O desenvolvimento do pensamento computacional possui alguns benefícios, como: 

        a resolução de problemas; 

        o estímulo à criatividade e à autonomia; 

        o incentivo ao conhecimento interdisciplinar.

- Vamos resolver um problema?
    
    Com o conhecimento adquirido nesta aula, vamos solucionar um problema utilizando os fundamentos do pensamento computacional.

 
    Imagine que você precisa estudar para uma avaliação. Primeiro, vamos decompor o problema. Se é necessário estudar para uma prova, você pode construir um cronograma de estudo, estudar todos os assuntos da prova e resolver exercícios para praticar o assunto.


    Depois, com a rotina de estudos estabelecida, você vai conseguir reconhecer padrões nos assuntos e nos exercícios. Isso permite que você abstraia os elementos que não têm relevância para a prova.

    
    Depois dessas etapas, é possível montar uma sequência de passos para ser bem sucedido na prova, ou seja, executar um algoritmo para resolver o problema.

- Pensamento computacional
    
    Assim, é possível concluir que os pilares do pensamento computacional podem ajudar na compreensão de problemas diversos a fim de encontrar soluções de forma mais rápida.
